Operator Guide

How To Use This POS

Fast steps for billing, inventory and reports.

1. Login & First-Time Setup

Create the first Admin user on new install, then login.

  • First install shows Setup screen for Admin username + password.
  • After setup, login is required every time the app opens.
  • Admin can create cashier users from Admin section.
  • Use Logout to switch users safely.

2. Billing (Barcode & Cart)

Scan items, review cart, and keep the counter fast.

  • Press Ctrl+F to focus barcode input, then scan.
  • Scanning the same barcode increases quantity automatically.
  • Use X to remove items; Edit/Return if needed.
  • Out-of-stock items are blocked.

3. Discount, Loyalty & Payment

Apply offers, redeem points, and take payment.

  • Discount cannot exceed subtotal.
  • Select customer to redeem loyalty points.
  • Cash/UPI/Card/Split supported; cash auto-fills total.
  • Complete sale, then print thermal or A4 invoice.

4. Products & Stock

Maintain accurate catalog and inventory.

  • Enter barcode/SKU, unit, pack size, price and stock.
  • GST > 0 requires HSN (4�8 digits).
  • Use Auto Barcode if item has no code.
  • CSV import for bulk add; archive instead of delete.

5. Purchases & Supplier Stock-in

Record supplier invoices to increase stock.

  • Add supplier, enter invoice no and items.
  • Qty and cost price required; batch/expiry optional.
  • Save purchase to add stock; return reduces stock safely.

6. Reports & Refunds

View sales, GST, and refund records.

  • Filter by date range and payment mode.
  • Print daily summary; export GST/Refund CSV.
  • Refund/Return needs Admin PIN and updates stock.

7. Backup & Admin

Keep data safe and manage hardware.

  • Daily auto backup + manual backup supported.
  • Restore from list with Admin PIN; app restarts.
  • Hardware settings for printer/scanner/cash drawer.

8. Khata & Shortcuts

Track credit accounts and work faster.

  • Create khata accounts; add credit/payment entries.
  • Ledger shows balance and history for each customer.
  • Shortcuts: F2 Dashboard, F3 Billing, F4 Products, F5 Reports.
  • Ctrl+Enter checkout, Ctrl+P thermal, Ctrl+Shift+P A4.
